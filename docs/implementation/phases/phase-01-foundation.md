## Уточнение по тестам

На этом этапе не добавляем технологии ради галочки.  
Базовый starter уже идет с `PHPUnit`, этого достаточно для foundation-слоя.  
`Pest` не является обязательным требованием первого этапа.

## Что не делать на этом этапе

- не начинать бизнес-логику сотрудников и БЗ
- не переносить сразу весь HTML в рабочие экраны
- не добавлять Redis, очереди, S3 и AI-слой
- не усложнять auth и ACL раньше следующего этапа
