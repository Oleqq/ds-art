# Этап 08. AI-агент

## Цель

Добавить интеллектуальный слой поверх уже готового приложения.

## Результат этапа

После завершения этапа должно быть:

- агент помогает искать контент
- агент отвечает по статьям и связанным данным
- агент работает в рамках прав пользователя

## Предусловия

Этот этап нельзя начинать, пока не готовы:

- права доступа
- поиск
- структурированные статьи
- понятные сущности сотрудников, файлов и категорий

## Задачи

- [ ] определить use cases агента
- [ ] определить источники данных агента
- [ ] построить безопасный retrieval слой
- [ ] встроить ACL в агентный доступ
- [ ] ограничить ответы видимостью пользователя
- [ ] определить интерфейс агента внутри приложения
- [ ] добавить логирование обращений к агенту

## Запрет

Нельзя делать AI-функцию, которая читает всю систему без проверки прав.
