# Этап 06. Права доступа, поиск и стабилизация

## Цель

Закрыть безопасность базы знаний не только визуально, но и на backend-уровне: через Laravel policies, контроллеры, SQL-выборки и тесты.

Главное правило: UI может скрывать кнопки, но реальное ограничение доступа должно жить в backend.
