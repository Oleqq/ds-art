# Этап 06. Sidebar Search Preview

- В левом sidebar input поиска переведен из статичного submit в живой preview-режим.
- Пока пользователь вводит запрос, под полем поиска динамически подгружаются найденные разделы и статьи.
- Поиск идет через отдельный JSON endpoint preview, а Enter по-прежнему открывает полный экран поиска по базе знаний.
- Preview закрывается по `Escape`, по клику вне блока, по очистке input и после перехода по найденному результату.
- Для UX добавлены анимация появления, состояние загрузки и отдельная кнопка перехода на полный экран поиска.
