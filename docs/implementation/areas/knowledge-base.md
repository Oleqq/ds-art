# Область: База знаний

## Цель

Реализовать структуру и сценарии базы знаний из HTML-прототипа.

## Что входит

- главная страница БЗ
- дерево категорий
- подкатегории
- экран категории
- список статей внутри категории
- breadcrumbs
- фильтрация видимости по правам

## Обязательные действия

- создать категорию
- изменить категорию
- удалить категорию
- открыть статью
- создать статью
- сохранить статью
- удалить статью
- дублировать статью
- переместить статью
- копировать ссылку на статью

## Особенности

- дерево должно поддерживать порядок
- статья должна знать свою категорию
- пользователь должен видеть только доступные разделы

## Важный принцип

Сначала делаем корректную доменную структуру БЗ.

Нельзя строить статью как “кусок HTML где-то в поле” без понимания:

- кто автор
- где лежит статья
- кому она доступна
- как ее искать
